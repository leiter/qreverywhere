QR Everywhere - Mag-scan at Lumikha ng QR Code

Ang iyong all-in-one na kasama para sa QR code. I-scan ang mga umiiral na code kaagad o lumikha ng sarili mo sa loob ng ilang segundo.

NGAYON SA 25 WIKA
Available sa Ingles, Espanyol, Pranses, Aleman, Italyano, Portuges, Ruso, Hapon, Koreano, Tsino (Pinasimple at Tradisyonal), Arabe, Hindi, Bengali, Indones, Vietnamese, Thai, Turko, Polish, Dutch, Ukrainian, Filipino, Malay, Hebrew, at Swahili.

MAG-SCAN NG QR CODE
• Gamitin ang iyong camera para sa instant na pag-scan
• Mag-import ng mga larawan mula sa iyong photo library
• Mga babala sa kaligtasan ng URL para sa mga kahina-hinalang link
• Sinusuportahan ang lahat ng karaniwang format ng QR code

LUMIKHA NG QR CODE
• Text at URL - Ibahagi ang mga link at mensahe
• WiFi Network - Hayaang kumonekta kaagad ang mga bisita
• Contact Card (vCard at MeCard) - Ibahagi ang iyong mga detalye
• Email - Pre-filled na address na may subject line
• Numero ng Telepono - One-tap na tawag
• Calendar Event - Ibahagi ang mga imbitasyon sa pulong
• Lokasyon - GPS coordinates na may label
• App Store Link - iOS at Android app link
• Payment Link - PayPal at Venmo request
• Cryptocurrency - Bitcoin at Ethereum address

AYUSIN AT PAMAHALAAN
• Buong kasaysayan ng mga na-scan at nilikhang code
• Maghanap sa iyong kasaysayan ng QR code
• I-save at ibahagi ang iyong mga QR code anumang oras
• Widget sa home screen para sa mabilis na access

PRIVACY MUNA
• Hindi kailangan ng account
• Walang pangongolekta ng data o pagsubaybay
• Lahat ay nananatili sa iyong device

Simple. Mabilis. Pribado. Sa iyong wika.
